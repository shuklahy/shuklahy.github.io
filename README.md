# Hitendra Shukla - Portfolio Website

This repo contains the code for my personal portfolio website hosted by GitHub Pages. [Visit the live site](https://shuklahy.github.io/).

## Development

### Prerequisites

- Node.js (for local server)
- Git

### Testing Locally

To test changes locally before deploying:

1. Clone the repository:
   ```bash
   git clone https://github.com/shuklahy/shuklahy.github.io.git
   cd shuklahy.github.io
   ```

2. Start a local development server:
   ```bash
   # Using Node.js serve
   npx serve .

   # Or using Python (if available)
   python3 -m http.server 8000
   ```

3. Open your browser and navigate to `http://localhost:5000` (for serve) or `http://localhost:8000` (for Python).

4. Make your changes and refresh the browser to see updates.

### Deploying Changes

Since this is a GitHub Pages site deployed from the main branch:

1. Commit your changes:
   ```bash
   git add .
   git commit -m "Your commit message"
   ```

2. Push to the main branch:
   ```bash
   git push origin main
   ```

3. GitHub Pages will automatically build and deploy your changes. The site will be live at https://shuklahy.github.io/ within a few minutes.
